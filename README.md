# PCRediveWikiEventHidder

本项目为Chrome扩展程序，作用于https://pcredivewiki.tw/

允许用户自动隐藏进行中的活动列表，并添加一个按钮以切换显示/隐藏

这样就不会每次想使用别的功能时都要向下划好久的页面了

## 安装方法

请Clone本项目或下载Zip解压后使用Chrome的开发者模式载入本插件。
